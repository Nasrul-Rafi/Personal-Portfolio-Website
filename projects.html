Projects

This page summarizes the projects I have completed and published on my GitHub profile. Most of my earlier works are responsive landing pages, while my latest works include accessibility, adaptive learning, full-stack development and neural-network-based music generation.

Current Thesis

My thesis is currently in Phase 3. The title is:

An AI-Driven Multimodal Adaptive Learning Framework for Visually Impaired Learners

This research connects AI, adaptive learning, multimodal feedback and accessibility. It is related to my interest in building systems that will help visually impaired learners to understand and learn graphical content more effectively.

MERN Project

VI-Learn: Adaptive Accessible Micro-Learning Website

A full-stack adaptive learning platform designed for visually impaired learners. It supports course discovery, enrollment, lesson reading in small sections, accessibility controls, browser-based text-to-speech, quizzes with hints, progress tracking, rule-based adaptive support, interaction logging and dashboard analytics.

Tools/Focus: React, Vite, Tailwind CSS, DaisyUI, Node.js, Express.js, MongoDB, JWT, bcrypt

GitHub Repository

CSE425 / Neural Networks Project

Unsupervised Neural Network for Multi-Genre Music Generation

A CSE425 group project that implements unsupervised deep learning models for music generation using MIDI datasets. The system learns musical patterns such as melody, rhythm and harmony, then generates new music sequences. The repository includes preprocessing, model training, evaluation and generation modules.

Tools/Focus: Python, PyTorch, NumPy, Pandas, PrettyMIDI, Music21, Matplotlib, Scikit-learn, Autoencoder, VAE, Transformer
