# Personal Homepage Project

This project is a personal homepage created for my CSE 391 assignment. The main purpose of this website is to introduce myself, my academic background, my interests, my skills, my thesis work and some of my previous projects.

In this website, I added my personal information, education details, completed and ongoing courses, technical skills, hobbies and interests, project highlights and a contact form. I have also included my own local image and one web image to fulfill the image requirement. The website has internal navigation links, external links, a responsive table, hover effects and a footer that shows the page location and last modified date.

I also added separate pages for my skills and projects. The skills page explains my course-based technical skills such as Python, object-oriented programming, data structures, algorithms, database systems, artificial intelligence, machine learning and neural networks. The projects page contains brief descriptions of my previous GitHub projects.

The website is built using only basic web technologies:

* HTML
* CSS
* JavaScript
